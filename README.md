# Control de Gastos Personales

Aplicación web para el control y seguimiento de gastos personales. Desarrollada con React, TypeScript y Material-UI.

## Características

- Registro e inicio de sesión de usuarios
- Panel de control con resumen de gastos
- Gráficos para visualizar tus gastos
- Categorización de gastos
- Interfaz intuitiva y responsiva

## Requisitos Previos

- Node.js (v16 o superior)
- npm o yarn

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/control-gastos-personales.git
   cd control-gastos-personales
   ```

2. Instala las dependencias:
   ```bash
   npm install
   # o
   yarn install
   ```

## Cómo Ejecutar el Proyecto

Para iniciar el servidor de desarrollo:

```bash
npm run dev
# o
yarn dev
```

Luego, abre tu navegador y ve a [http://localhost:5173](http://localhost:5173)

## Construcción para Producción

Para crear una versión optimizada para producción:

```bash
npm run build
# o
yarn build
```

## Despliegue

Puedes desplegar este proyecto en cualquier servicio de hosting que soporte aplicaciones Node.js, como Vercel, Netlify o Heroku.
